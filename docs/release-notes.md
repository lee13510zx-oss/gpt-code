# koreastudy 릴리스 노트

## 0.9.0-local-rc

무료 정적 웹앱 릴리스 후보입니다.

### 추가된 기능

- 학교급/학년/과목/단원 선택
- 요약, 표, 마인드맵, 문제 생성
- 문제 카테고리 4종
- 문제 형식 5종
- 교과서 정보 입력
- 직접 입력 기반 생성
- 학습목표와 성취기준 메모
- 과목군별 학습 팁과 출제 포인트
- 보관함
- 오답노트, 복습 완료, 개별 삭제
- 학습지 보기와 인쇄/PDF 저장
- 텍스트 저장
- 다크모드
- 데이터 검증 화면
- 무료 배포 준비 화면
- 품질 체크리스트와 자가진단
- 마지막 승인 대시보드
- Gemini 선택 연결 구조
- Netlify Functions 키 보호 구조
- API 키 없이 작동하는 무료 규칙/샘플 기반 모드

### 제한 사항

- Netlify 공개 배포 상태는 Netlify 계정의 Deploys 화면에서 확인해야 합니다.
- Gemini API 키 등록은 선택 사항이며, 나이 제한이나 계정 제한이 있으면 사용하지 않습니다.
- 교과서 원문 기반 요약과 시험지 분석은 저작권 검토 전까지 제한됩니다.
- 공식 성취기준 원문은 아직 공식 자료 대조 전입니다.
