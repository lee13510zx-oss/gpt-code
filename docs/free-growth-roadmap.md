# koreastudy 무료 성장 로드맵

## 1단계: 무료 정적 웹앱

- HTML, CSS, JavaScript만 사용한다.
- Netlify 무료 플랜 또는 GitHub Pages에서 배포한다.
- 브라우저 localStorage로 보관함을 제공한다.
- Gemini 키가 없어도 샘플 데이터로 작동한다.
- 무료 범위의 코드, 문서, UI 변경은 별도 승인 없이 계속 진행한다.

## 2단계: 무료 AI 연결

- Gemini 무료 티어를 사용한다.
- 개인 테스트에서는 브라우저 저장 키를 허용한다.
- 공개 배포에서는 Netlify Functions와 환경변수를 사용해 키를 숨긴다.
- 요청 실패, 한도 초과, 네트워크 오류 시 샘플 데이터로 fallback한다.

## 3단계: 데이터 품질 확대

- 초등, 중등, 고등의 과목별 단원 시드를 늘린다.
- 2015 개정과 2022 개정의 차이를 별도 데이터로 분리한다.
- 과목별 출력 규칙을 강화한다.
- 수학은 공식과 유형, 과학은 법칙과 실험, 영어는 어휘와 독해, 국어는 작품/문법/쓰기, 사회는 자료 해석을 강화한다.

## 4단계: 무료 배포 자동화

- GitHub 저장소에 코드를 올린다.
- Netlify 무료 플랜과 연결한다.
- 배포 환경변수에 `GEMINI_API_KEY`를 등록한다.
- 배포 후 앱이 키 없이도 작동하는지, 키가 있을 때 AI 요청이 되는지 점검한다.

## 5단계: 수익 이후 유료 전환 후보

아래 항목은 수익이 생기고 무료 한도 때문에 실제 문제가 생겼을 때만 검토한다.

- 유료 AI API 한도 확대
- 관리형 DB
- 사용자 계정과 클라우드 저장
- 공식 교과서/콘텐츠 라이선스
- 검색 인덱스
- 서버 로그와 분석 도구
