# koreastudy 최종 인수 문서

## 현재 상태

koreastudy는 무료 정적 웹앱 릴리스 후보 상태입니다.

외부 계정, 유료 서비스, 저작권 허가가 필요한 작업을 제외하고 앱 내부 기능, 문서, 배포 준비 구조는 모두 포함되어 있습니다. GitHub 저장소 `lee13510zx-oss/gpt-code`에는 앱 파일 업로드를 진행했습니다.

## 실행 파일

브라우저에서 아래 파일을 열면 앱이 실행됩니다.

```text
index.html
```

## 주요 기능

- 초등학교, 중학교, 고등학교 학교급/학년/과목/단원 선택
- 요약, 표, 마인드맵, 문제 생성
- 문제 카테고리와 문제 형식 선택
- 교과서 출판사, 저자, 단원, 직접 입력 내용 기반 생성
- 학습목표, 성취기준 메모, 학습 팁, 출제 포인트 표시
- 보관함 저장
- 오답노트 저장, 복습 완료, 개별 삭제
- 학습지 보기와 인쇄/PDF 저장
- 텍스트 저장
- 데이터 검증 상태, 배포 준비 상태, 품질 체크리스트, 최종 승인 화면
- Gemini 선택 연결 구조와 Netlify Functions 키 보호 구조
- Gemini 키가 없거나 발급할 수 없거나 실패할 때 무료 규칙/샘플 데이터 fallback

## 승인 전까지 하지 않은 것

- Gemini API 키 발급 또는 등록
- 나이 제한이나 계정 제한이 있는 Gemini 키 발급 우회
- 유료 API, 유료 DB, 유료 서버 사용
- 교과서 원문, 시중 시험지, 사설 문제집 무단 저장
- 공식 성취기준 원문 확정 반영

## 마지막 승인 후 순서

1. `docs/deferred-approvals.md`를 확인합니다.
2. Netlify 무료 플랜에서 GitHub 저장소를 연결합니다.
3. Gemini를 공개 서비스에 연결하려면 Netlify 환경변수 `GEMINI_API_KEY`를 등록합니다.
4. `docs/deploy-checklist.md`와 `docs/qa-checklist.md`에 따라 수동 QA를 진행합니다.
5. 앱 안의 `승인`, `배포`, `품질`, `검증` 화면을 확인합니다.

## 운영 원칙

- 무료 범위의 변경은 계속 진행 가능
- 유료 전환은 사용자 수익 발생 후 검토
- 저작권 있는 콘텐츠는 권리 확인 전까지 저장하지 않음
- AI 결과는 학습 보조 자료이며 최종 검토가 필요함
