# koreastudy QA 체크리스트

무료 배포 전 로컬 브라우저와 Netlify 배포 URL에서 확인할 항목입니다.

## 기본 화면

- 품질 화면에서 자가진단 실행 버튼을 누르면 필수 요소와 데이터 상태가 표시된다.
- 홈 화면이 열린다.
- 상단 탭이 모두 전환된다.
- 하단 내비게이션이 모바일 폭에서 겹치지 않는다.
- 다크모드가 적용되고 새로고침 후 유지된다.

## 학습 자료 생성

- 학교급, 학년, 과목, 단원 선택이 작동한다.
- 요약, 표, 마인드맵, 문제 화면이 같은 선택값으로 갱신된다.
- 학습목표와 성취기준 메모가 표시된다.

## 문제와 오답

- 문제 카테고리 4종이 작동한다.
- 문제 형식 5종이 작동한다.
- 해설 보기/숨기기가 작동한다.
- 오답노트 저장과 전체 삭제가 작동한다.
- 오답노트 복습 완료 버튼이 복습 횟수와 최근 복습 시간을 갱신한다.
- 오답노트 개별 삭제가 작동한다.

## 저장과 출력

- 보관함 저장과 전체 삭제가 작동한다.
- 텍스트 저장이 작동한다.
- 학습지 보기와 인쇄/PDF 저장이 작동한다.
- 학습 팁과 출제 포인트가 요약, 학습지, 텍스트 저장에 포함된다.

## AI와 배포

- Gemini 키가 없어도 fallback으로 결과가 생성된다.
- Netlify 환경변수 등록 후 함수 요청이 작동한다.
- 배포 화면과 검증 화면이 현재 상태를 표시한다.
