# koreastudy

무료로 시작하는 초중고 AI 학습 정리 웹앱입니다. 빌드 도구 없이 `index.html`만으로 실행되며, 나중에 GitHub와 Netlify 무료 플랜으로 배포할 수 있게 구성했습니다.

## 포함 기능

- 초등학교, 중학교, 고등학교 전체 선택 구조
- 주요 과목, 예체능/생활 과목, 고등 선택과목 포함
- 학년/과목/단원 기반 요약, 표, 마인드맵, 문제 생성
- 단원 선택 기반 생성
- 문제 카테고리: 개념 확인, 내신 대비, 모의고사 대비, 서술형/수행평가
- 문제 형식: 혼합형, 객관식, OX, 빈칸, 주관식
- 교과서 출판사, 저자, 단원명, 직접 입력 내용 기반 생성
- 브라우저 localStorage 보관함
- 브라우저 localStorage 오답노트, 복습 완료, 개별 삭제
- 생성 결과 텍스트 파일 저장
- 학습지 보기와 인쇄/PDF 저장
- 데이터 검증 상태, 배포 준비 상태, 앱 자체 QA 체크리스트
- 필수 화면/데이터/localStorage 자가진단
- Gemini API 키는 선택 사항
- Gemini 키를 발급할 수 없으면 무료 규칙/샘플 기반 모드로 작동
- 나중에 키 사용이 가능해지면 Netlify Functions + `GEMINI_API_KEY` 환경변수 기반 AI 요청
- Gemini 요청 실패 또는 키 미입력 시 샘플 데이터 fallback

## 무료 운영 원칙

- 유료 서버, 유료 DB, 유료 API 사용 없음
- 교과서 원문과 시중 시험지를 무단 저장하지 않음
- 사용자가 권리가 있는 자료만 직접 입력하도록 안내
- 공개 배포 전에는 API 키를 Netlify Functions 환경변수로 숨기는 것을 권장
- 무료 범위의 코드, 문서, UI, 데이터 변경은 사전 승인된 것으로 보고 진행

## 실행 방법

`index.html`을 브라우저에서 열면 바로 실행됩니다.

권장 확인 순서:

1. 홈에서 학교급, 학년, 과목, 단원을 선택합니다.
2. `학습 자료 생성`을 누릅니다.
3. 요약, 표, 마인드맵, 문제, 오답노트, 학습지 화면을 확인합니다.
4. `품질` 화면에서 자가진단을 실행합니다.

## Netlify 배포

이 폴더 전체를 Netlify에 업로드하거나 GitHub 저장소에 올린 뒤 Netlify 무료 플랜으로 연결하면 됩니다.

빌드 명령은 필요 없습니다.

배포 디렉터리:

```text
outputs/koreastudy
```

Gemini 키는 선택 사항입니다. 나이 제한이나 계정 제한으로 발급할 수 없다면 이 단계는 건너뜁니다.

나중에 Gemini 키를 숨겨서 쓰려면 Netlify 환경변수에 아래 값을 추가합니다.

```text
GEMINI_API_KEY=발급받은_Gemini_API_Key
```

환경변수가 없으면 앱은 브라우저에 저장된 키를 사용하려고 시도하고, 키가 없거나 요청이 실패하면 무료 규칙/샘플 데이터로 작동합니다.
